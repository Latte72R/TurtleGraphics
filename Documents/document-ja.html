日本語

English

はじめに¶

Turtle graphicsとは、1967年にWally Feurzeig、Seymour Papert、Cynthia Solomonによって開発されたLogoで紹介された一般的な幾何学的描画ツールの実装です。

Turtleは、単純な動きを繰り返すプログラムを使って複雑な形を描くことができます。

Turtle graphicsは、物理的な "turtle"（ペンを持った小さなロボット）が床に置かれた紙に絵を描く様子を表現しています。

即座に目に見える形でフィードバックが得られるため、学習者がプログラミングの概念やソフトウェアとのインタラクションに出会うための効果的で実績のある方法です。 また、一般的にグラフィカルな出力へのアクセスも便利です。

Turtleはもともと、教室で教師が使うための教育ツールとして作られました。 グラフィカルな出力を必要とするプログラマーにとっては、より複雑な外部ライブラリを導入することなくより複雑な外部ライブラリを導入する必要がないです。

チュートリアル¶

初めての方はここから始めてください。このチュートリアルでは、Turtleの基本的な描き方をいくつか紹介します。

基本的な描き方¶

Turtleを100進めます：

await turtle.forward(100);

Turtleが東に向かって線を引いているのが見えるはずです。Turtleの向きを変え、左（反時計回り）に120度回転するようにします：

await turtle.left(120);

続けて三角形を描いてみましょう：

await turtle.forward(100);
await turtle.left(120);
await turtle.forward(100);

Turtleが操作するにつれてさまざまな方向を向くことに注目してみましょう！

Penの制御¶

色（例えばcolor('blue')）と線の幅（例えばwidth(3)）を変えてからもう一度描いてみましょう。

また、Turtleを移動させる前にup()でPenを持ち上げれば、 描画せずにTurtleを移動させることもできます。再び描画を開始するには、down()を使用します。

Turtleの位置¶

Turtleをスタート地点に戻す（画面外に消えてしまった場合に有効です）：

await turtle.home();

Turtleのメソッドと対応する関数¶

このセクションのほとんどの例は、turtleと呼ばれるTurtleインスタンスを参照しています。

Turtleの移動¶

await turtle.forward(distance)¶
Parameters

distance – 整数

Turtleが向かっている方向に、指定された距離distanceだけTurtleを前進させる。

await turtle.forward(25);
await turtle.backward(distance)¶
Parameters

distance – 整数

Turtleが向かっている方向とは逆方向に、指定された距離distanceだけTurtleを後方に移動させる。Turtleの向きは変わりません。

await turtle.backward(30);
await turtle.right(angle)¶
Parameters

angle – 整数

Turtleを指定された角度angleだけ右向きに回転させる。

await turtle.right(45);
await turtle.left(angle)¶
Parameters

angle – 整数

Turtleを指定された角度angleだけ左向きに回転させる。

await turtle.left(45);
await turtle.goto(x, y)¶
Parameters

  • x – 整数

  • y – 整数

Turtleを座標(x, y)に移動させます。ペンが下りていれば線を引きます。Turtleの向きは変化しません。

await turtle.goto(60,30);
await turtle.setx(x)¶
Parameters

x – 整数

Turtleの第１座標をxにします。第２座標は変化しません。

await turtle.setx(10);
await turtle.sety(y)¶
Parameters

y – 整数

Turtleの第２座標をyにします。第１座標は変化しません。

await turtle.sety(-10);
await turtle.setheading(to_angle)¶
Parameters

to_angle – 整数

Turtleの向きを角度to_angleに設定します。以下はよく使われる方向を角度で表わしたものです：

標準モード

0 - east

90 - north

180 - west

270 - south

 
await turtle.setheading(90);
await turtle.home()¶

Turtleを原点（座標(0,0)）に移動し、方向を開始方向に設定する。

await turtle.home();
await turtle.circle(radius, extent)¶
Parameters

  • radius – 整数

  • extent – 整数

与えられた半径radiusで円を描きます。円の中心はTurtleから半径radiusの距離だけ左にあります。 中心角extentが与えられなければ、デフォルトで完全な円になります。 中心角extentが完全な円でない場合、円弧の一方の端点は現在のペンの位置になります。 半径radiusが正の場合は反時計回りに、そうでない場合は時計回りに円弧を描きます。 Turtleの方向はextentの分だけ変化します。

await turtle.circle(120, 180);  # draw a semicircle
turtle.dot()¶
Parameters

  • size – 自然数

与えられた直径sizeで円形の点を描く。

turtle.dot(20);
turtle.stamp()¶

現在のTurtleの位置にTurtleの形状のコピーをつくります。

turtle.stamp();
turtle.speed(speed)¶
Parameters

speed – 整数（０～１０）

Turtleの速度を０〜１０の範囲の整数値に設定します。

以下に速度の対応表を示します：

速度対応表

0 - 最速

10 - 高速

6 - 標準

3 - 低速

1 - 超低速

スピードは1から10まであり、線画と亀の回転のアニメーションがどんどん速くなります。

※注意：speed = 0はアニメーションが行われないことを意味します。 forward/backはTurtleをジャンプさせ、同様にleft/rightはTurtleを瞬時にターンさせます。

turtle.speed(8);

描画状態¶

turtle.pendown()¶

ペンを下ろします – 動くと線が引かれます。

turtle.penup()¶

ペンを上げます – 動いても線は引かれません。

turtle.pensize(width)¶
Parameters

width – 自然数

>>> 
turtle.pensize(10);   # from here on lines of width 10 are drawn

色の制御¶

turtle.pencolor(*args)¶

ペンの色を設定します

３種類の入力形式が受け入れ可能です：

pencolor(colorstring)

ペンの色をcolorstringに設定します、その値は "yellow"や "#33cc8c"のような色指定文字列です。

pencolor((r, g, b))

ペンの色を r, g, b のタプルで表された RGB の色に設定します。

pencolor(r, g, b)

ペンの色を r, g, b で表された RGB の色に設定します。

Turtleの輪郭は新しく設定されたペンの色で描かれます。

>>> 
turtle.pencolor("brown");
turtle.pencolor((0.2, 0.8, 0.55));
turtle.pencolor('#32c18f');
turtle.fillcolor(*args)¶

塗りつぶしの色を設定します

３種類の入力形式が受け入れ可能です：

fillcolor(colorstring)

塗りつぶしの色をcolorstringに設定します、その値は "yellow"や "#33cc8c"のような色指定文字列です。

fillcolor((r, g, b))

塗りつぶしの色を r, g, b のタプルで表された RGB の色に設定します。

fillcolor(r, g, b)

塗りつぶしの色を r, g, b で表された RGB の色に設定します。

Turtleの内部は新しく設定された塗りつぶしの色で描かれます。

>>> 
turtle.fillcolor("violet");
turtle.fillcolor((50, 193, 143));  # Integers, not floats
turtle.fillcolor('#ffffff');
turtle.color(*args)¶

ペンの色と塗りつぶしの色を設定します

３種類の入力形式が受け入れ可能です：

以下のように１～３の引数を使用したいくつかの入力フォーマットが認められています：

color(colorstring), color((r,g,b)), color(r,g,b)

pencolor() と同様に入力し、fillcolorとpencolorの両方を指定された値に設定します。

color(colorstring1, colorstring2), color((r1,g1,b1), (r2,g2,b2))

pencolor(colorstring1)および fillcolor(colorstring2)と等価であり、 他の入力フォーマットが使われる場合も同様です。

Turtleの輪郭と内部が、新しく設定された色で描かれます。

>>> 
turtle.color("red", "green");
turtle.color("#285078", "#a0c8f0");

塗りつぶし¶

turtle.begin_fill()¶

塗りつぶされる図形を描画する直前に呼び出します。

turtle.end_fill()¶

begin_fill() を呼び出した後に描画された図形を塗りつぶします。

交差する多角形や複数の図形の重なり領域が塗りつぶされるかどうかは、オペレーティングシステムのグラフィックス、重なりのタイプ、重なりの数に依存します。

>>> 
turtle.color("black", "red");
turtle.begin_fill();
await turtle.circle(80);
turtle.end_fill();

Turtleの可視性¶

turtle.hideturtle()¶

Turtleを見えなくします。Turtleを隠すと描画が速くなるので、複雑な描画をしている最中にこれを行うのは良いアイデアです。

>>> 
turtle.hideturtle();

>>> 
turtle.showturtle();

turtle.shapesize(10);

特別なTurtleのメソッド¶

turtle.undo()¶

最後に行われたTurtleの操作を取り消します。

for (let i = 0; i < 4; i++) {
    turtle.forward(50);
    turtle.left(90);
}
for (let i = 0; i < 8; i++) {
    turtle.undo();
}
turtle.reset()¶

画面からTurtleの描画を削除し、Turtleの中心を合わせ直し、Turtleに関する変数をデフォルト値に設定します。

turtle.reset();
await turtle.sleep(secs)¶
Parameters

  • secs – 正の数（小数または整数）

指定された秒数の間、呼び出し元のスレッドの実行を一時停止します。

システム内の他のアクティビティとのスケジューリングのため、中断時間は要求された時間よりも長くなる可能性があります。

await turtle.sleep(3.0);
